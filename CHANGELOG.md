# Changelog

All notable changes to this project will be documented here.

---

## [1.0.0] – 2026-03-28

### Added
- Sidebar soundboard view with category grouping
- 2-column button grid per category
- Loop support for ambient sounds (green glow indicator)
- Master volume slider
- Stop All button
- Per-sound settings: label, sub-label, file, icon, loop, volume
- Custom icon support (emoji + image files from vault)
- Icon picker modal
- Sound editor modal with move-to-category support
- Settings tab with full CRUD for categories and sounds
- Persistent storage via Obsidian data API
- Desktop-only flag
